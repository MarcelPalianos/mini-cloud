# Mini Cloud (DevOps Lab)

Multi-container DevOps project using Docker Compose.

## Stack
- Nginx (web server)
- Flask (backend)
- PostgreSQL (database)
- Redis (cache)
- Grafana (monitoring)

## Features
- Containerized services
- Service-to-service communication
- Persistent database storage
- Custom backend connected to DB

## Run

```bash
docker compose up -d --build
